# Kazakh Punctuation Restoration

This project builds a machine learning model that restores punctuation in Kazakh text generated by Automatic Speech Recognition (ASR) systems.

The project was developed during the **WISH AI/ML Hackathon organized by NU ACM-W Student Chapter**.

## Problem

Automatic Speech Recognition systems usually output text without punctuation, which makes the text difficult to read and harder for downstream NLP systems to process.

Example input:

сәлем менің атым бақыт қалың қалай

Expected punctuation labels:

COMMA O O PERIOD O QUESTION

Rendered sentence:

Сәлем, менің атым Бахыт. Қалың қалай?

The task is to predict which punctuation mark (if any) should follow each word.

## Task

This is a **token classification problem** where the model predicts one of the following labels for each word:

- **O** — no punctuation  
- **COMMA** — comma after the word  
- **PERIOD** — sentence-ending period  
- **QUESTION** — question mark  

Exclamation marks are treated as **PERIOD**.

## Model

The model is based on **XLM-RoBERTa**, a multilingual transformer model designed for cross-lingual natural language processing.

The model was fine-tuned for **token classification** to predict punctuation labels for each word in a sequence.

## Dataset

The model was trained using:

- **KazPunct Hackathon dataset (Kaggle)**
- **Hugging Face Datasets**

The dataset contains Kazakh text sequences with punctuation labels for each token.

## Evaluation

The model is evaluated using **Macro F1 score** over the punctuation classes:

- COMMA
- PERIOD
- QUESTION

The **O label is excluded from scoring** because it dominates the dataset.

Example evaluation code:

```python
from sklearn.metrics import f1_score

score = f1_score(
    y_true,
    y_pred,
    labels=['COMMA','PERIOD','QUESTION'],
    average='macro'
)
```
